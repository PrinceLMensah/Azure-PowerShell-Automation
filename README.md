# Azure PowerShell Automation Project

## Overview
This PowerShell script automates the deployment of a **Windows Server 2022 VM in Azure** while applying **security best practices**:
✅ Deploys an Azure VM  
✅ Configures **Network Security Groups (NSG)**  
✅ Enables **Azure Backup for disaster recovery**
